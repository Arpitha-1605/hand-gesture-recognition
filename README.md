# ✋ Hand Gesture Control System 🎯

A real-time hand gesture recognition system to control media and volume using computer vision.

---

## 🚀 Features

- 👍 Play / Pause media
- ✊ Pause
- ☝️ Next track
- ✌️ Previous track
- 🤏 Volume control using finger distance

---

## 🧠 Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- PyAutoGUI
